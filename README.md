# Anime Search App

Vite + React 18 + TypeScript SPA using Jikan API (MyAnimeList).

## Quickstart

1. Install dependencies (use npm):

```bash
npm install
```

2. Start dev server (runs on port 4000):

```bash
npm run dev
```

3. Open http://localhost:4000
